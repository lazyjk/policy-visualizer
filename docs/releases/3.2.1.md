## What's New in 3.2.1

### Frontend — Service Picker Filter & Sort

The service picker now includes controls above the service list when two or
more services are present in the uploaded file.

- **Type filter** (`All types` / `RADIUS` / `TACACS` / `RADIUS PROXY`):
  limits the visible list to services matching the selected `service_type`.
- **Sort** (`Default` / `A → Z` / `Z → A`): sorts the filtered list
  alphabetically using locale-aware comparison.
- Controls reset automatically when a new file is uploaded.
- A "No services match." message is shown if the active filter excludes all
  services.
- **`frontend/src/components/UploadPanel.tsx`**

### Normalizer — Internal Reference Token Display

`expr_to_node_label` now prefers `displayValue` over `raw` when the raw RHS
value is a ClearPass internal reference token of the form `Namespace:NNN`
(e.g. `NadGroup:3004`). Previously only bare numeric IDs and numeric CSV
values triggered this substitution.

- **`src/normalizer.py`**: added `_INTERNAL_REF` regex to `_pick_rhs`.
- **`tests/test_normalizer.py`**: 1 new test
  (`test_node_label_uses_display_for_internal_ref_rhs`).
