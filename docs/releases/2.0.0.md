# Policy Visualizer Release Notes — 2.0.0

## Release

- Version: `2.0.0`
- Release date: `2026-02-25`
- Release type: `ga`
- Commit/tag: `v2.0.0`

## Summary

First public GA release. Delivers a fully interactive, browser-based flow diagram tool
for network policy XML exports. The complete compiler pipeline (XML → Policy IR → Flow IR
→ React Flow diagram) ships with export, annotation, and UX features built across the 2.0
alpha/beta/RC cycle.

## Highlights

- Full pipeline: XML upload → service picker → interactive diagram in the browser
- PNG, PDF, and SVG export (client-side, no backend route required)
- Annotation nodes: sticky notes with inline editing, connectable with dashed arrows
- Per-shape node color picker (6 node types)
- Snap-to-grid toggle
- RADIUS and TACACS service types both supported
- 71-test suite passing; deterministic output verified
- Docker-packaged: `docker compose up` is the complete deployment

## Scope for This Release

- Included (full 2.0 feature set):
  - XML parser: RADIUS and TACACS services, all enforcement profile types
  - Condition normalizer: 14-operator canonical Boolean AST
  - Policy IR: stable deterministic IDs, fail-fast unresolved reference detection
  - Flow IR compiler: directed graph with `rank_group` hooks for future collapse
  - FastAPI backend: `POST /api/flow`, `POST /api/services`, `GET /api/health`
  - React Flow frontend: dagre layout, two-pass sizing, chain compression, overlap resolution
  - Export: PNG / PDF / SVG (client-side via `html-to-image` and `jspdf`)
  - Annotations: sticky-note nodes with dashed-arrow connectors
  - Node color picker and snap-to-grid toggle
  - Graphviz CLI path retained for offline/scripted use
  - Docker packaging: API on :8000, Nginx/React on :80
- Not included (explicitly deferred):
  - UX-230: Session drag persistence
  - UX-231: Auto-align action
  - UX-233: Multi-service tabbed viewing
  - COL-240/241: Vertical chain collapse/expand
  - ANN-223: WYSIWYG annotation editor
  - ISE integration (moved to 3.0 track)

## Features

### Export
- PNG export via `html-to-image` — captures full diagram at any zoom/pan state
- PDF export via `jspdf` — wraps full-diagram PNG
- SVG export — vector format, white or transparent background
- Transparency toggle: white vs transparent background for all export formats

### Diagram UX
- Annotation nodes: add sticky notes anchored to diagram context; editable in place
- Dashed annotation edges: connect annotation nodes to flow nodes
- Per-shape node color picker: 6 node types (start, decision, process, action, end, annotation)
- Snap-to-grid toggle for alignment during manual layout adjustments
- NodeToolbar tooltips: full condition text on hover for long decision node labels

### Policy/Compiler Pipeline
- RADIUS and TACACS service parsing from XML export
- Canonical Boolean AST normalization (14 operators)
- Stable deterministic Policy IR with UUID-based IDs
- Directed Flow IR graph with `rank_group` hints for future collapse

### API
- `POST /api/flow` — compile service from XML, return Flow IR JSON
- `POST /api/services` — list services in uploaded XML (service picker)
- `GET /api/health` — liveness check
- Error responses: 413 oversized, 415 wrong extension, 422 invalid XML or unresolved refs

## Fixes
