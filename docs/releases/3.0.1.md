# Policy Visualizer Release Notes — 3.0.1

## Release

- Version: `3.0.1`
- Release date: `2026-03-04`
- Release type: `patch`
- Commit/tag: `v3.0.1`

## Summary

Patch release with annotation editor improvements, a normalizer display-value fix,
handle visibility cleanup, and MIT license added.

## Fixes

- **Normalizer — numeric RHS display preference** (`src/normalizer.py`):
  `expr_to_node_label` now prefers `displayValue` over `rhs_raw` when the raw
  value is a bare numeric ID or numeric CSV (e.g. RADIUS `NAS-Port-Type = 19`
  now renders as `Wireless-802.11 (19)` in decision node labels).

## Improvements

### Annotation Editor

- **Text alignment**: Left / Center / Right toolbar buttons added to
  `AnnotationNode` editor (`frontend/src/components/nodes/nodeTypes.tsx`).
- **Text color picker**: Color swatch input added to toolbar; applies inline
  color to selected text.
- **Larger font sizes**: 28 pt and 36 pt added to the font-size dropdown.
- **Removed redundant handles**: Extra source handles (top / right / bottom /
  left) removed from `AnnotationNode`; target handle remains for layout pinning.
- **Box-shadow removed** from annotation node container.

### Diagram Canvas

- **Handle visibility** (`frontend/src/index.css`): Connection handles on all
  non-annotation nodes are now hidden (`opacity: 0; pointer-events: none`) to
  reduce visual noise. Edges continue to render correctly.

### Project

- **MIT license added**: `LICENSE` file created; `frontend/package.json` now
  includes `"license": "MIT"`; README updated with license section.
