# Policy Visualizer Release Notes — 3.3.0

## Release

- Version: `3.3.0`
- Release date: `2026-03-11`
- Release type: `minor`
- Commit/tag: `v3.3.0`

## Summary

Minor release delivering three features: a Policy Details Panel that surfaces
rule-level conditions in a side inspector when a node is selected, a PDF
appendix export that appends policy rule detail tables to the exported PDF,
and a redesigned Export panel with a unified format picker and single Export CTA.
Also includes an ISE fix for policy set match condition display.

---

## What's New in 3.3.0

### Policy Details Panel

A new side panel (`PolicyDetailsPanel`) appears when the user clicks a decision
or action node in the diagram. It shows the full rule condition (formatted
predicates, AND/OR combinators) and action details for the selected rule,
sourced from a new `/api/flow` `include_details` response field.

- **`frontend/src/components/PolicyDetailsPanel.tsx`** — new panel component
- **`src/policy_details.py`** — backend module: condition serialization and
  rule detail extraction for ClearPass and ISE pipelines
- **`api/schemas.py`** — `PolicyDetails` / `RuleDetail` schemas added
- **`api/routes/flow.py`** — `include_details` support wired into flow endpoint
- **`frontend/src/components/FlowDiagram.tsx`** — node click → panel open wiring
- **`frontend/src/App.tsx`** / **`frontend/src/api.ts`** — panel state + fetch integration

### PDF Appendix Export

When PDF format is selected and policy details are loaded, an **Include appendix**
checkbox appears in the Export panel. Checking it exports a multi-page
`{serviceName}-appendix.pdf` containing: service match context, authentication
rules, role mapping / authorization rules, enforcement / profile application rules,
and warnings (when present). Condition text is formatted in monospace; sections
include automatic page breaks and per-page headers.

- **`frontend/src/components/FlowDiagram.tsx`** — `addAppendixPages()`, "Include
  appendix" checkbox toggle, `handleExportPdfWithAppendix()` handler

### Export UX — Unified Format Picker (option-a)

The Export panel has been redesigned with a single **Export** button and a
format picker (PNG / SVG / PDF / Draw.io). Replaces the previous per-format
buttons. Options (transparent background, grid) remain available.

- **`frontend/src/components/FlowDiagram.tsx`** — ExportPanel redesigned
